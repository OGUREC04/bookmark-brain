# БТ-10: Onboarding и команды

> Обновлено: 2026-05-24

## Назначение и границы

Покрывает: первый контакт с ботом (`/start`), скрытую регистрацию пользователя,
приветствие новому vs вернувшемуся, общий каталог команд, справку (`/help`),
часовой пояс (`/tz`), переключатель режима (`/silent`), вход в Mini App и его
навигацию.

НЕ покрывает: детали каждой команды по содержанию — они в профильных БТ
(захват, списки, напоминания, поиск, тихий режим, уборка). Здесь — точка входа
и ссылки.

## Пользовательские истории

Как пользователь:
1. Я хочу написать боту `/start` и сразу понять, что он умеет, без чтения мануала.
2. Я хочу, чтобы повторный `/start` не сыпал тем же длинным приветствием каждый раз.
3. Я хочу, чтобы бот завёл мне аккаунт сам — без регистрации, форм, паролей.
4. Я хочу один раз задать свой часовой пояс, чтобы «напомни завтра в 9» сработало по моим часам.
5. Я хочу открыть приложение прямо из бота одной кнопкой и листать заметки визуально.
6. Я хочу видеть полный список команд через `/help`, даже те, что скрыты из меню Telegram.

## Проблема / зачем так

У бота нет экрана регистрации — и не должно быть. Пользователь приходит из
Telegram, у него уже есть identity. Первое же сообщение (или `/start`) молча
создаёт аккаунт на бэкенде через `X-Bot-Secret`, и человек этого не замечает.
Это намеренно: трение на входе убивает second-brain быстрее всего.

Раньше приветствие было одно на все случаи — вернувшийся юзер получал тот же
«вот что я умею» с нуля при каждом `/start`. Сейчас бот помнит, что уже здоровался
(флаг `onboarding_welcomed` в настройках), и второй раз отвечает коротко.

Часовой пояс — дефолт `Europe/Moscow`. Без него юзер из Калининграда получал бы
напоминание на час раньше, поэтому `/tz` существует, но не навязывается: большинство
в MSK, дефолт их устраивает.

## Поведение (правила)

### Регистрация (скрытая)

Аккаунт создаётся **лениво** при первом контакте — не отдельным шагом. `ensure_user`
на любом входящем сообщении/коллбэке дёргает `POST /auth/bot` (заголовок
`X-Bot-Secret`). Бэкенд: нет юзера с таким `telegram_id` → создаёт; есть → обновляет
профиль и `last_active`. Возвращает JWT (живёт 7 дней, бот кэширует на 6).

Дефолты нового юзера: `timezone = Europe/Moscow`, `silent_mode = true` (тихий режим
по умолчанию), `settings = {}`, `bookmarks_count = 0`.

### `/start` — новый vs вернувшийся

| | Новый (флаг `welcomed` не стоит) | Вернувшийся (флаг стоит) |
|---|---|---|
| Текст | Длинное приветствие: что умеет, «попробуй переслать сообщение», базовые команды | Короткое: «С возвращением, {имя}! Я на месте. Перешли что-нибудь или /list, /search, /random, /help» |
| Кнопка Mini App | «📚 Открыть приложение» (если задан `MINI_APP_URL`) | та же кнопка |
| Побочный эффект | ставит флаг `onboarding_welcomed` | ничего |

Имя берётся из `first_name` Telegram; если пусто — «друг».

### Каталог команд

Зарегистрированы (`set_my_commands` + хендлеры). В меню Telegram показываются не все
— `/clean`, `/reprocess`, `/clearlists`, `/clearreminders`, `/tz` скрыты, но работают
(перечислены в `/help`).

| Команда | В меню | Что делает | Детали |
|---|---|---|---|
| `/start` | — (служебная) | приветствие + регистрация + кнопка Mini App | этот БТ |
| `/help` | ✅ | полная справка по всем возможностям | этот БТ |
| `/todo` | ✅ | создать список задач (`/todo пункт1, пункт2`) | [БТ-4](бт-04-списки-задач.md) |
| `/lists` | ✅ | показать только списки задач | [БТ-4](бт-04-списки-задач.md) |
| `/unpin` | ✅ | открепить все списки (сами списки остаются) | [БТ-4](бт-04-списки-задач.md) |
| `/remind` | ✅ | создать напоминание (`/remind текст время`) | [БТ-5](бт-05-напоминания.md) |
| `/reminders` | ✅ | активные напоминания + история, reply-управление | [БТ-5](бт-05-напоминания.md) |
| `/list` | ✅ | все закладки (5 на страницу, пагинация) | [БТ-7](бт-07-поиск-просмотр.md) |
| `/search` | ✅ | семантический поиск (`/search запрос`) | [БТ-7](бт-07-поиск-просмотр.md) |
| `/random` | ✅ | случайная закладка | [БТ-7](бт-07-поиск-просмотр.md) |
| `/stats` | — | статистика по закладкам | [БТ-7](бт-07-поиск-просмотр.md) |
| `/silent` | ✅ | переключить тихий/обычный режим (`on`/`off`) | [БТ-8](бт-08-тихий-режим.md) |
| `/tz` | — | показать/сменить часовой пояс | этот БТ |
| `/clean` | — | удалить мои сообщения за последние 48ч | [БТ-9](бт-09-уборка.md) |
| `/clearlists` | — | архивировать все списки (с подтверждением) | [БТ-9](бт-09-уборка.md) |
| `/clearreminders` | — | отменить все активные напоминания | [БТ-9](бт-09-уборка.md) |
| `/reprocess` | — | переобработать закладку AI заново | [БТ-2](бт-02-ai-обработка.md) |

Неизвестная команда (`/foobar`) под команды не подходит и проваливается в общий
catch-all `start` — обрабатывается как обычный текст (пойдёт в захват/AI), а не
ошибка «нет такой команды».

### Часовой пояс (`/tz`)

| Ввод | Реакция |
|---|---|
| `/tz` | показывает текущий пояс + список популярных IANA-имён |
| `/tz Europe/Kaliningrad` | меняет на указанный IANA-пояс |
| `/tz reset` | возвращает дефолт `Europe/Moscow` |
| `/tz Мухосранск` (не IANA) | бэкенд → 400, бот: «не похож на IANA-имя» + подсказка |

Дефолт для всех — `Europe/Moscow (UTC+3)`. Валидация: бэкенд проверяет имя через
`ZoneInfo`, неизвестное → 400.

### Mini App — вход и навигация

Открывается двумя путями: кнопка «📚 Открыть приложение» под `/start` и постоянная
menu-кнопка «📚 Открыть» в Telegram (обе — `WebAppInfo(MINI_APP_URL)`; если URL не
задан в конфиге — кнопок нет).

Авторизация в Mini App **отдельная** от бота: фронт берёт Telegram `initData`,
шлёт `POST /auth/telegram`, получает свой JWT. Регистрация и здесь ленивая — тот же
`get_or_create` по `telegram_id`. Если `initData` протух (TTL ~24ч в долго открытом
WebView), фронт делает один авто-retry, при повторном 401 → `AuthExpiredError`
(«Сессия истекла. Перезапусти Mini App из бота»).

Первый экран — вкладка **«мысли»** (лента закладок). Навигация: нижний бар с тремя
вкладками + центральная кнопка `+`.

| Элемент | Назначение |
|---|---|
| мысли | лента закладок (стартовый экран) |
| пространства | папки/коллекции |
| я | профиль + статистика + настройки |
| `+` (центр FAB) | быстрое создание заметки (sheet) |
| поиск | НЕ вкладка — оверлей поверх «мыслей» |

Экран «я»: аватар (инициал), хэндл (`@username` или имя), счётчик закладок, дата
регистрации. Настройки (тихий режим, AI-подсказки, таймзона, тема, экспорт) пока
read-only — тап показывает тост «в разработке».

## Use Cases

### UC-10.1. Первый `/start`

- **Предусловия:** юзера нет в БД (или флаг `welcomed` не стоит).
- **Триггер:** `/start`.
- **Основной сценарий:**
  1. `ensure_user` создаёт аккаунт на бэкенде (дефолты: MSK, silent=true).
  2. Бот шлёт длинное приветствие с именем + кнопку «📚 Открыть приложение».
  3. Ставит флаг `onboarding_welcomed`.
- **Альтернативные сценарии:**
  - A1. `MINI_APP_URL` не задан → приветствие без кнопки.
  - A2. Бэкенд недоступен → «Ошибка подключения к серверу. Попробуй позже», аккаунт не создан.
  - A3. У юзера пустой `first_name` → имя = «друг».
- **Результат:** аккаунт есть, флаг welcomed стоит.

### UC-10.2. Повторный `/start`

- **Предусловия:** флаг `welcomed` уже стоит.
- **Триггер:** `/start`.
- **Основной сценарий:**
  1. Бот видит флаг → короткое «С возвращением, {имя}!».
  2. Кнопка Mini App та же.
  3. Флаг не трогает, приветствие не дублирует.
- **Результат:** короткий ответ.

### UC-10.3. Скрытая регистрация на первом сообщении

- **Предусловия:** юзера нет, он шлёт не `/start`, а сразу контент.
- **Триггер:** пересланное сообщение / ссылка / голосовое.
- **Основной сценарий:**
  1. `ensure_user` создаёт аккаунт прозрачно (без приветствия).
  2. Сообщение уходит в обычный пайплайн захвата.
- **Правила:** регистрация не требует `/start`; `/start` лишь добавляет приветствие.

### UC-10.4. Установка часового пояса

- **Триггер:** `/tz Asia/Yekaterinburg`.
- **Основной сценарий:**
  1. Бэкенд валидирует имя через `ZoneInfo`.
  2. Сохраняет, бот: «Часовой пояс: Asia/Yekaterinburg».
  3. Все будущие напоминания разбираются в этом поясе.
- **Альтернативные сценарии:**
  - A1. `/tz` без аргумента → показывает текущий + популярные пояса.
  - A2. `/tz reset` → возвращает `Europe/Moscow`.
  - A3. Невалидное имя → 400 → «не похож на IANA-имя» + подсказка, пояс не меняется.
- **Результат:** `user.timezone` обновлён или показан.

### UC-10.5. Открытие Mini App

- **Предусловия:** задан `MINI_APP_URL`.
- **Триггер:** тап «📚 Открыть приложение» (или menu-кнопка).
- **Основной сценарий:**
  1. Telegram открывает WebView, фронт берёт `initData`.
  2. `POST /auth/telegram` → JWT (аккаунт создаётся, если ещё нет).
  3. Грузится вкладка «мысли».
- **Альтернативные сценарии:**
  - A1. `initData` протух → авто-retry; при повторном 401 → экран «Сессия истекла, перезапусти из бота».
  - A2. Юзер впервые открывает Mini App, не нажав `/start` в боте → `auth/telegram` создаёт аккаунт, приветствия нет.
- **Результат:** открыта лента закладок.

### UC-10.6. Навигация в Mini App

- **Триггер:** тапы по нижнему бару.
- **Основной сценарий:**
  1. «мысли» → лента; «пространства» → папки; «я» → профиль.
  2. `+` → sheet быстрого создания.
  3. Иконка поиска → оверлей поиска (нативный Back закрывает его).
- **Альтернативные сценарии:**
  - A1. Тап по read-only настройке на «я» → тост «в разработке».

### UC-10.7. `/help`

- **Триггер:** `/help`.
- **Основной сценарий:** бот шлёт полную справку — сохранение, поиск, списки,
  напоминания, режимы, уборка, включая скрытые из меню команды.
- **Результат:** одно сообщение-справка.
