# BookmarkBrain — Mini App / iOS UI Kit

Single Expo codebase serves both Telegram Mini App and iOS. Components mock the high-fidelity look; not production-wired.

## Components
- `Feed.jsx` — main scrollable feed with "Brain suggests" slot
- `BookmarkCard.jsx` — compact + expanded states
- `SuggestionCard.jsx` — proactive AI suggestions in feed
- `SearchBar.jsx`, `TagChip.jsx`, `AiBadge.jsx`, `TaskListCard.jsx`
- `BottomTab.jsx` — tab bar
- `MiniApp.jsx` — app shell, demonstrates click-thru
