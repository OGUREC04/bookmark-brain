# Mini App Redesign — Brief & PRD draft

**Статус:** v0 draft на основе брифа 2026-05-15.
**Контекст:** текущий Mini App работает, но перегружен и не отражает новый функционал (Reminders). Юзер хочет переделать почти с нуля. Целевая аудитория — те, «кому впадлу делать заметки» → интерфейс лёгкий, понятный, минималистичный, без потери функционала.

---

## Терминология (КРИТИЧНО)

| Старое | Новое | Причина |
|---|---|---|
| Закладки | **Мысли** | Звучит легче. Психологически проще для аудитории «лень» |
| Папки | **Пространства** (включают папки внутри) | Унификация с будущими Smart Spaces (Phase 5) |
| Bookmark | Thought (UI-уровень, БД не трогаем) |

**Bot commands не меняем** (всё ещё `/list`, `/search`). Меняется только UI-копия в Mini App.

---

## Главный сценарий (90% запусков)

1. Юзер открывает Mini App из бота
2. Видит список «Мысли» — обзор накопленного
3. Видит шапку с 🔔 (если есть активные напоминания)
4. **Не ищет старое** часто — старое ценно связями (Phase 5/6), а не доступом
5. Может тапнуть на карточку → детали → редакт

**Вторичные сценарии:** разобрать накопленное (move в пространство), поставить reminder, удалить мусор.

## Архитектура навигации

```
┌─────────────────────────────────┐
│ BookmarkBrain         🔔  ⋮     │ ← шапка: логотип, колокольчик (reminders), меню
├─────────────────────────────────┤
│                                 │
│  [Список / Поиск / Пространства]│ ← контент таба
│                                 │
│                                 │
│                          ┌───┐  │
│                          │ + │  │ ← FAB создания (текст/голос)
│                          └───┘  │
├─────────────────────────────────┤
│  💭 Мысли  🔍 Поиск  ▦ Простр.  │ ← bottom nav (3 таба)
└─────────────────────────────────┘
```

**Табы:**
1. **💭 Мысли** — все закладки + фильтры (теги / избранное / тип)
2. **🔍 Поиск** — отдельный экран с AI-summary
3. **▦ Пространства** — Smart Spaces + традиционные папки внутри них

**Колокольчик (🔔):** counter активных напоминаний. Тап → список ближайших reminders.

**FAB «+»:** открывает быстрый ввод (text input + voice button). MVP — только текст. Голос отложим (есть в боте).

## Карточка «Мысли»

```
┌─────────────────────────────────────────┐
│ 📄  Заголовок мысли          ⭐  11 мая │
│     Краткое summary в одну строку...    │
│     #тег1 #тег2                         │
└─────────────────────────────────────────┘
```

**Состав:**
- **Иконка типа** слева (lucide / heroicons style) — НЕ эмодзи. Цвет приглушённый.
  - 🔗 Link / 📄 Article / 🎤 Voice / 📋 Task list / 💡 Idea / ✓ Action
- **Заголовок** (1 строка, truncate)
- **Summary** (1 строка, truncate). Опциональная.
- **Дата** справа сверху, мелкая, приглушённая
- **⭐** если избранное, справа сверху рядом с датой
- **Теги** мелким шрифтом, приглушённым цветом, под summary

**НЕ показываем на карточке:**
- ❌ Цветные бейджи типов (`action`, `article`) — это сейчас и это шум
- ❌ Все теги списком (>3 — обрезка с «...»)
- ❌ Большие отступы между элементами карточки

**Бейджи-индикаторы (если применимо):**
- ⏰ + дата — если на закладке стоит reminder в ближайшие 7 дней
- ✓ 3/5 — task-list с прогрессом

## Quick Actions (long-press на карточке)

Меню:
- **🗑 Удалить** — с подтверждением. **Learning signal для Phase 4:** «удалил быстро после save = плохо обработали»
- **⭐ В избранное** / **☆ Убрать**
- **▦ В пространство** → bottom-sheet со списком
- **⏰ Напомнить** → выбор времени (today / завтра / через неделю / custom)

**Swipe-актион:** TBD (можно left=delete, right=archive — обсудить с a11y-architect)

## Избранное — Anti-Junk механизм

**Проблема:** избранное со временем превращается в свалку.

**Решение:**
1. Видно как фильтр-чип ⭐ в Мыслях (нет отдельного таба)
2. **Месячный review-пинг от бота:** «В избранном 15 мыслей, 8 не открывались >30 дней. Проверить?»
3. Открывает специальный экран с stale-favorites → юзер свайпает «оставить / убрать»
4. Подсветка stale карточек в обычной выдаче (мягкий индикатор)

## Учётные сигналы (для Phase 4 Learning)

Фиксируем в БД (не сейчас, но архитектурно учесть):

| Событие | Сигнал |
|---|---|
| Удаление в <1 час после save | «классификация плохая» → корректировка few-shots |
| Удаление stale (>30 дней без открытия) | «не нужно вообще» → не учим |
| Перемещение из auto-space в другое | «routing плохой» → корректировка прompt-а пространства |
| Снятие избранного через review-пинг | «временный сигнал, не баг» |

## Стиль — ФИНАЛ (после 4 итераций мокапов 2026-05-15)

**Базовая палитра — Echo / Sage (из дизайн-системы `frontend/src/styles/tokens.css`):**
- `--bg-page: #F5F1EB` — cream paper
- `--bg-surface: #FAF7F1` / `--bg-elev: #FFFCF6`
- `--fg-1: #2C2825` — warm ink
- `--fg-2: #6B645B` / `--fg-3: #9A938A`
- `--border-1: #E8E1D2`
- `--brand: #7A9C7A` (Sage — **закреплён**)
- `--brand-tint: #E2EDE2` / `--brand-deep: #2F4A2F`

**Типографика:**
- **Onest** (sans, open-source Söhne-альтернатива, Cyrillic-first, Sana AI vibe)
- Fallback: `-apple-system, BlinkMacSystemFont, "SF Pro Text", system-ui`
- НЕТ italic-акцентов (Cormorant Garamond не используем)
- Letter-spacing tight (`-0.02em` на крупных)

**Иконки:**
- **Lucide** outline 1.75–1.9 stroke
- НЕТ эмодзи в UI (только в копии сообщений бота, не в Mini App)
- Звезда favorite: Lucide Star filled, амбер `#E0A82E`

**Layout — двойной (юзер выбирает через toggle):**
- **A · Лента (Feed)** — один вертикальный поток, карточки разной высоты, padding 14×16, radius 18
- **B · TG-чаты** — плотный список с круглой иконкой-«аватаром» 48×48, title + preview-1-line + time + badge

**Шапка:**
- Логотип «Мысли» 28px / 700 / tight tracking
- Справа: view-toggle (grid/list) + уведомления `[🕒 3]` pill в sage tint

**Категории (tabs):**
- Горизонтальный chip-row, sticky, paint в стиле «папки TG»
- Активный: filled `--fg-1` / `--bg-page`
- Counter справа от названия (`Все · 49`)

**Bottom nav — Liquid Glass:**
- Floating капсула, отступ 14px по бокам и 18px снизу
- `backdrop-filter: blur(28px) saturate(180%)`
- Semi-transparent white over content (~62%)
- 5 пунктов: Мысли / Поиск / **+** (центр) / Пространства / Я
- FAB: 46×46 круг, тёмный `rgba(44,40,37,0.92)`, **без brand-цвета**
- Active item: `rgba(0,0,0,0.05)` фон, текст `--fg-1`

**Карточка Лента:**
- Top row: `[icon · type-label]` слева, `[★] [date]` справа
- Title 16px / 600 / -0.015em
- Summary 14px / 400 / `--fg-2`
- Tasklist: progress-bar + сами задачи с ✓/○ зачёркнутыми
- Reminder badge: sage-tint pill с lucide-clock

**Карточка TG-чат:**
- Круглая иконка-«аватар» 48×48, цветной background по типу (sage / amber / sky / dust / rose)
- Title 15px / 600 inline со звездой если favorite
- Preview одной строкой: `<b>тип ·</b> текст` (приглушённый)
- Time справа (12px, tabular-nums)
- Badge-counter (`3/5` для tasklist, число для reminders) или badge-pin

**Финальные мокапы:** `tools/mockups/miniapp-pinterest.html`

## Технический хвост

- **BrowserRouter → HashRouter** (Telegram WebView надёжнее с #)
- **Theme params** — подхватить `Telegram.WebApp.themeParams` для цветов
- **Cache-busting** — после деплоя нужен hard close (learned instinct)
- **Auth 401** — добавить refresh JWT через initData
- **Loading states** — skeleton-loaders вместо spinner (легче ощущение)
- **Empty states** — короткое CTA «отправь сообщение боту»
